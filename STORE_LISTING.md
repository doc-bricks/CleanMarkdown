## Deutsch

### Kurzbeschreibung (max 100 Zeichen)
Lokaler Markdown-Viewer und -Editor mit Lesemodus, PDF-Export und ruhiger Oberfläche.

### Beschreibung (max 10.000 Zeichen)
CleanMarkdown ist ein schneller lokaler Markdown-Viewer und -Editor für Menschen, die klare Textarbeit statt schwerer Editor-Suiten wollen. Die App öffnet Markdown-Dateien direkt vom Dateisystem, zeigt eine ruhige Leseansicht und bietet einen fokussierten Raw-Editor ohne WYSIWYG-Ballast.

**Was CleanMarkdown kann:**

- Ruhiger Lesemodus für Markdown-Dateien mit sauberer Vorschau
- Raw-Markdown-Editor mit Struktur-Buttons für Überschriften, Listen, Tabellen, Links, Bilder und Fußnoten
- PDF-Export aus derselben lokalen Ansicht
- Helles und dunkles Theme
- Deutsche und englische Oberfläche
- Autosave mit einstellbarem Intervall
- Optionale Scroll-Synchronisierung zwischen Lesen und Editor
- Leichtgewichtige Mathe-Vorschau für `$...$`, `$$...$$`, `\(...\)` und `\[...\]`
- Relative Bilder und lokale Assets werden passend zum geöffneten Dokument geladen

**Für wen ist CleanMarkdown gedacht?**

Für alle, die Markdown-Dateien lokal lesen, prüfen, korrigieren und exportieren möchten: Autoren, Entwickler, Wissensarbeiter, Studierende und alle, die strukturierte Textdateien ohne Cloud-Zwang pflegen.

**Warum CleanMarkdown statt eines großen Editors?**

- Fokus auf einen klaren Einzeldatei-Workflow statt auf Projekt-Overhead
- Keine Konten, kein Cloud-Zwang, keine Telemetrie
- Sofort verständliche Oberfläche mit nur zwei Hauptansichten
- Lokal und datensparsam: Dateien bleiben auf dem eigenen Gerät

## English

### Short Description (max 100 chars)
Local Markdown viewer and editor with reading mode, PDF export and a calm interface.

### Description (max 10,000 chars)
CleanMarkdown is a fast local Markdown viewer and editor for people who want clear text work instead of heavy editor suites. It opens Markdown files directly from the file system, shows a calm reading view, and provides a focused raw editor without WYSIWYG overhead.

**What CleanMarkdown does:**

- Calm reading mode for Markdown files with a clean preview
- Raw Markdown editor with helper buttons for headings, lists, tables, links, images and footnotes
- PDF export from the same local view
- Light and dark theme
- German and English UI
- Autosave with configurable interval
- Optional scroll sync between reading and editor
- Lightweight math preview for `$...$`, `$$...$$`, `\(...\)` and `\[...\]`
- Relative images and local assets resolve from the opened document folder

**Who is CleanMarkdown for?**

Anyone who wants to read, review, correct and export Markdown files locally: writers, developers, knowledge workers, students, and anyone maintaining structured text files without a cloud dependency.

**Why choose CleanMarkdown instead of a larger editor?**

- Focused single-file workflow instead of project overhead
- No accounts, no cloud requirement, no telemetry
- Simple interface with only two core views
- Local and privacy-friendly: your files stay on your own device
