# Verhaltenskodex / Code of Conduct

## Deutsch

### Unser Versprechen

Wir möchten rund um CleanMarkdown eine respektvolle, hilfreiche und konstruktive Zusammenarbeit fördern. Alle Menschen sollen sich beim Melden von Fehlern, Vorschlagen von Ideen und Einreichen von Änderungen sicher fühlen können.

### Erwartetes Verhalten

- Freundlicher und sachlicher Umgangston
- Konstruktive Rückmeldungen statt persönlicher Angriffe
- Offenheit für unterschiedliche Erfahrungsstände
- Fokus auf das Produkt und seine Verbesserung

### Nicht akzeptabel

- Beleidigungen, Herabwürdigungen oder Einschüchterung
- Diskriminierende oder belästigende Aussagen
- Öffentliche Preisgabe privater Informationen anderer Personen
- Bewusst destruktives Verhalten in Diskussionen oder Pull Requests

### Durchsetzung

Bitte melde problematisches Verhalten nicht öffentlich eskalierend. Nutze bei GitHub die vorhandenen Moderations- und Meldewege oder kontaktiere den Repository-Eigentümer direkt über GitHub.

## English

### Our Pledge

We want CleanMarkdown to have a respectful, helpful, and constructive collaboration culture. Everyone should feel safe when reporting bugs, proposing ideas, or submitting changes.

### Expected Behavior

- Friendly and professional communication
- Constructive feedback instead of personal attacks
- Respect for different experience levels
- Focus on improving the product

### Unacceptable Behavior

- Insults, harassment, intimidation, or discrimination
- Publishing private information about others
- Deliberately destructive behavior in discussions or pull requests

### Enforcement

Please do not escalate incidents publicly if avoidable. Use GitHub moderation/reporting tools where available or contact the repository owner directly through GitHub.

Basierend auf dem Geist des Contributor Covenant v2.1.
